# GitHub Publishing Guide / GitHub 发布指南

## Recommended publishing steps

```bash
git init
git add .
git commit -m "Initial personal finance Excel BI template"
git branch -M main
git remote add origin https://github.com/<your-username>/personal-finance-excel-bi-template.git
git push -u origin main
```

## Before making the repository public

- Remove any real bank statements, screenshots or private financial data.
- Keep only anonymized examples.
- Review workbook metadata if privacy is important.
- Decide whether to add an open-source license.

## 中文提醒

公开仓库前，请检查是否包含真实个人流水、银行截图、税务文件、账户余额或住址信息。建议公开模板与匿名示例，把真实数据放在私有仓库或本地文件夹。
