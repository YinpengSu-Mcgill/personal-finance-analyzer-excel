# Monthly Maintenance Workflow / 月度维护流程

## English

1. Open the template.
2. Add new expense records to `Expense_Input` / `总支出`.
3. Add new income records to `Income_Input` / `总收入`.
4. Update `Input_Settings` / `输入设置` if cash balance or emergency-fund assumptions changed.
5. Update `Sales_Tax_Settings` / `税费设置` only when tax assumptions change.
6. Review Dashboard, Monthly Summary, Category Summary and Investable Cash.
7. Export a monthly backup copy if you want a locked archive.

## 中文

1. 打开模板。
2. 在 `Expense_Input` / `总支出` 追加新的支出记录。
3. 在 `Income_Input` / `总收入` 追加新的收入记录。
4. 如果现金余额或备用金假设变化，更新 `Input_Settings` / `输入设置`。
5. 如果消费税假设变化，再更新 `Sales_Tax_Settings` / `税费设置`。
6. 查看总览、每月总结、各类别总结和可投资资金。
7. 如需留档，可以每月另存一个归档副本。
