# Roadmap: From Excel Template to Web App

## Phase 1: Excel Project

Current version:

- Bilingual Excel templates
- Formula hub architecture
- Dashboard and summary pages
- Formula documentation

## Phase 2: Browser-Based Prototype

Planned features:

- Upload `.xlsx` or `.csv`
- Parse data locally in the browser
- Normalize transaction records
- Generate dashboard charts
- Calculate emergency fund and investable cash
- Export summary as CSV

Recommended stack:

- React
- Vite
- Tailwind CSS
- SheetJS / xlsx
- Recharts

## Phase 3: Advanced Web App

Potential enhancements:

- Column mapping screen
- Duplicate transaction detection
- Custom category rules
- Bilingual UI
- Financial health score
- Downloadable report

## Privacy Principle

All uploaded financial data should be processed locally in the browser. No financial records should be sent to a server.
