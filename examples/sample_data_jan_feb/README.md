# January–February Sample Data / 1–2 月示例数据

This folder contains CSV exports of the workbook's first two months of sample data.

这个文件夹保存了模板中第一个月与第二个月的示例输入数据。

Files:

| File | Language | Description |
|---|---|---|
| `expense_input_jan_feb_zh.csv` | Chinese | Expense input sample |
| `income_input_jan_feb_zh.csv` | Chinese | Income input sample |
| `expense_input_jan_feb_en.csv` | English | Expense input sample |
| `income_input_jan_feb_en.csv` | English | Income input sample |

Use these records as formatting examples. Replace descriptions, categories and amounts with your own personal data when using the workbook.

这些数据用于展示录入格式。实际使用时，可以把描述、类别和金额替换为自己的个人数据。
